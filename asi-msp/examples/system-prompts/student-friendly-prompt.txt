SECURITY CONCEPTS TO TEACH:
- Port: A "door" that services use to communicate
- Service: A program running on the target (like SSH, HTTP)
- Vulnerability: A weakness that could be exploited
- CVE: A unique ID for a known security issue
- SQL Injection: Tricking a database into running malicious commands
- Brute Force: Trying many passwords to guess the right one
